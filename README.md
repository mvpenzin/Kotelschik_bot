# SNT-Bot "КОТЕЛЬЩИК"

Автоматизированная система для садоводческого товарищества СНТ "Котельщик", включающая Telegram-бота для садоводов и веб-панель управления для администрации.

## Основные возможности

### Telegram-бот для жителей
- **Профиль**: Просмотр и редактирование ФИО, номера телефона и даты рождения. Управление списком своих участков.
- **Информационный блок**:
  - Актуальный прогноз погоды в СНТ.
  - Расписание электричек.
  - Список важных контактов администрации и служб.
  - Действующие тарифы (членские взносы, электроэнергия).
- **Оплата**: Инструкции по оплате через Сбербанк и по QR-коду с автоматическим формированием назначения платежа.
- **Развлечения**: Ежедневные праздники, случайные цитаты, анекдоты и цитаты с Bash.im.
- **Автоматизация**: Автоматическое закрытие клавиатуры через 60 секунд бездействия для поддержания чистоты чата.

### Панель управления (Admin Dashboard)
- **Статус системы**: Мониторинг работы бота и времени аптайма.
- **Управление контактами**: Удобный интерфейс для добавления, редактирования и удаления контактов СНТ.
- **Логирование**: Просмотр истории действий бота и системных событий в реальном времени.

## Технологический стек

- **Бэкенд**: Go (Golang) с использованием фреймворков `echo` (API) и `telebot/v5` (Telegram API).
- **База данных**: PostgreSQL.
- **Фронтенд**: React + TypeScript + Vite.
- **Стилизация**: Tailwind CSS + shadcn/ui.
- **Управление состоянием**: TanStack Query (React Query).

## Настройка

Основные параметры бота настраиваются в файле `main.ini`:
- `kb_timeout`: Время (в секундах) до автоматического закрытия клавиатуры.

## Развертывание на Replit

1. Убедитесь, что установлена переменная окружения `DATABASE_URL` для подключения к PostgreSQL.
2. Установите секрет `TELEGRAM_BOT_TOKEN` с вашим токеном от BotFather.
3. Запустите проект через кнопку "Run" (используется рабочий процесс "Start application").
